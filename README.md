# 🔬 Smart Blood Cell Analyzer (AI-Powered)

![Project Interface](detection_screenshot.png)

## 🚀 Overview
An interactive medical AI application built with **Streamlit** and **YOLOv8** to automate blood cell analysis. The system detects and counts **Red Blood Cells (RBCs)**, **White Blood Cells (WBCs)**, and **Platelets** from microscopic images in real-time.

## ⚙️ Core Technical Features:
- **Object Detection:** Utilizes the YOLOv8 architecture for high-speed, accurate cell localization.
- **Automated Counting:** Instantly reports the total count of each cell type via an intuitive dashboard.
- **Robust Image Processing:** Handles various image formats (JPG, PNG) with automatic RGB conversion to ensure model compatibility.
- **Professional UI:** A clean, responsive web interface designed for laboratory and research use.

## 🛠️ Tech Stack:
- **Framework:** Streamlit (Python)
- **AI Engine:** YOLOv8 (Ultralytics)
- **Imaging:** OpenCV & Pillow
- **Model Hosting:** Trained on BCCD Dataset, optimized for fast inference.

## 🤖 Engineering Perspective:
As a Computer & Automatic Control Engineering student, I developed this system to demonstrate how **Computer Vision** acts as a powerful "visual sensor" in automated diagnostic systems, significantly reducing human error and time in medical labs.
