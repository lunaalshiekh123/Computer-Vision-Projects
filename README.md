# Computer Vision & Digital Image Processing

## Description
Built a computer vision system focused on enabling machines to interpret and process visual information from the world. The project involved applying image processing techniques and feature extraction to identify and analyze visual patterns.

## Tools used:
- **Language:** Python
- **Libraries:** OpenCV, NumPy, Matplotlib.

## Key Features:
- Digital Image Processing (Filtering, Edge Detection).
- Feature Extraction from visual data.
- Object/Pattern Recognition logic.
